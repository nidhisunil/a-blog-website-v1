A website to store your personal diary
Uses Node, EJS

Download node and npm and nodemon
Run "npm install" in terminal
Run "nodemon app.js"
Go to localhost:3000/ for the home page
Go to localhost:3000/contact for the contact page
Go to localhost:3000/about for the about page
To compose a new blog post, go to localhost:3000/compose and type something.
It will now be visible on the homepage
The website is not connected to a database, so the information you type will disappear on restarting the server
